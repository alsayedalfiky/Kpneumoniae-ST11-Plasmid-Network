# Project Overview: Our study defines the evolutionary trajectory of the epidemic ST11 lineage, highlighting:
The "Clonal Sweep": ST11 comprises 30% of the cohort but only 1.05% of phylogenetic diversity.
Pangenome Dynamics: Modeling an open pangenome (Heaps' Law alpha = 0.5919).
Plasmid Networks: Identification of "lineage-specific plasmotypes" and hub replicons (IncFII, IncFIB).
Genomic Stabilization: Chromosomal integration of key resistance genes, e.g., blaKPC-2 and blaCTX-M-15.
